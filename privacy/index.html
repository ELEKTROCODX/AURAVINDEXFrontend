Privacy Policy

Last Updated: April 12, 2025

Important Notice: AURA VINDEX is an academic project developed by ELEKTRO Team as part of a university simulation in El Salvador. It is not a formal company and has no commercial purposes. All features, including subscriptions and delivery services, are simulated. No financial data is collected and no real transactions are performed.

1. Information We Collect

We collect the following personal information when users create an account:

  • First name and last name
  • Email address
  • Gender
  • Date of birth
  • User-uploaded photo/avatar
  • Home address (only if requesting book delivery)

We do not collect phone numbers at the moment, but we may include this in the future for features such as 2FA via SMS. We do not collect IP addresses, device data, or sensitive personal data as defined by law.

We also maintain internal audit records that log user actions within the app, linked to their database ID.

2. When Data is Collected

Data is collected only through the registration form when creating an account. No information is stored if the user does not register.

3. How Data is Used

Your data is used for the following purposes:

  • To create and manage your account
  • To send account-related emails (e.g., password recovery)
  • To verify age eligibility based on date of birth
  • To deliver books to your home address (if requested)

4. Third Parties and Services

Currently, AURA VINDEX does not use third-party services to manage personal data. All data is stored securely on our own API server (AURAVINDEXAPI). We formerly used Nodemailer to send emails, and may update this in the future.

5. User Rights

Users have the right to:

  • Request the deletion of their personal data (name, email, gender, date of birth, avatar, address)
  • Modify or update their personal information via the profile editor

Audit logs will be preserved for security and monitoring purposes, associated only with the user's database ID after deletion.

6. Data Security

Data is hosted on our own secure cloud servers on Digital Ocean using Docker. Access is protected with SSH authentication, and passwords are encrypted. We currently do not perform data backups.

7. Contact Information

If you have any questions, please contact us at aura24vindex@gmail.com.

8. Age Requirement

Users must be at least 16 years old to register. If under 18, parental consent is required.

9. Policy Updates

We will notify users via email whenever changes are made to this privacy policy.
